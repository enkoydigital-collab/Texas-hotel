# GlobalHotel architecture

## Overview
GlobalHotel is structured as a modern Next.js front end with content-driven sections for booking, dining, dashboards, and guest flows.

## Core modules
- Marketing and content pages
- Room catalog and reservation UX
- QR dining experience
- Dashboard views

## Deployment approach
- Dockerfile for containerized frontend
- docker-compose.yml for local orchestration
- GitHub Actions for CI
